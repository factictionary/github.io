🎉 Mobile Menu Test Page

This page demonstrates the new mobile navigation menu implementation.

📱 How to Test

Desktop (≥768px)

  • You should see the regular horizontal navigation menu
  • Hamburger icon should be hidden
  • Resize browser window to less than 768px to see mobile menu

Mobile (<768px)

  • Click the hamburger icon (☰) in the top right
  • Menu should slide in from the right
  • Backdrop should appear behind the menu
  • Click backdrop or menu item to close
